Предобработка и анализ данных с использованием библиотеки Pandas

Введение

В современном анализе данных важным этапом является их предварительная обработка. Сырые данные часто содержат пропуски, ошибки и несоответствия, которые могут негативно повлиять на результаты анализа. Для решения этих задач используется библиотека pandas языка Python, предоставляющая удобные инструменты для очистки, трансформации и анализа данных.

Цель работы

Освоение методов очистки и трансформации данных с использованием библиотеки pandas на примере реального набора данных Titanic.

Описание набора данных

В работе использован датасет Titanic с платформы Kaggle. Данные содержат информацию о пассажирах корабля, включая:

  • PassengerId — идентификатор
  • Survived — факт выживания
  • Pclass — класс билета
  • Name — имя пассажира
  • Sex — пол
  • Age — возраст
  • SibSp — количество родственников (братья/сёстры/супруги)
  • Parch — количество родителей/детей
  • Ticket — номер билета
  • Fare — стоимость билета
  • Cabin — каюта
  • Embarked — порт посадки

Первичный анализ данных

Был выполнен анализ структуры данных:

  • Определены типы данных признаков
  • Обнаружены пропуски в столбцах Age, Cabin и Embarked
  • Получены статистические характеристики числовых признаков

Основные наблюдения:

  • Признак Cabin содержит большое количество пропусков
  • Возраст пассажиров имеет разброс и выбросы
  • Стоимость билета (Fare) имеет значительные отклонения

Визуализация данных

Построены гистограммы распределения числовых признаков и тепловая карта корреляций.

Выводы:

  • Распределение возраста близко к нормальному
  • Признак Fare имеет выбросы
  • Обнаружены зависимости между признаками

Обработка пропусков

Age

  • Среднее значение: 29.7
  • Медиана: 28

Пропуски заполнены медианой, так как она устойчива к выбросам.

Embarked

Пропуски заполнены наиболее частым значением (модой).

Cabin

Из признака извлечена первая буква, обозначающая палубу.

Трансформация данных

Были выполнены следующие преобразования:

  • Признак Pclass преобразован в категориальный (F, S, T)
  • Из имени выделен новый признак Title
  • Пол преобразован в числовой формат (0 — мужской, 1 — женский)
  • Создан признак FamilySize (размер семьи)
  • Создан бинарный признак IsAlone

Обработка выбросов

Для признака Fare: - Построен boxplot
- Выбросы определены методом IQR
- Применена winsorization (ограничение значений 95-м перцентилем)

Для возраста: - Построено распределение

Агрегация и анализ

Были рассчитаны агрегированные показатели:

  • Средняя выживаемость по классам
  • Группировка по классу и полу
  • Медианный возраст по портам посадки

Основные выводы:

  • Пассажиры 1 класса выживали чаще
  • Женщины имели значительно более высокий шанс выживания
  • Пассажиры, путешествующие в одиночку, выживали реже

Корреляционный анализ

Построена матрица корреляций.

Основные зависимости:

  • Пол имеет сильную корреляцию с выживаемостью
  • Стоимость билета положительно связана с выживанием
  • Размер семьи сильно коррелирует с SibSp и Parch
  • Признак IsAlone обратно коррелирует с FamilySize

Следует учитывать, что корреляция не означает причинно-следственную связь.

Метрики качества обработки

  • Процент заполненных данных: 87.5%
  • Уменьшено количество пропусков
  • Созданы новые информативные признаки

Заключение

В ходе работы были изучены методы предобработки данных с использованием библиотеки pandas. Были выполнены очистка данных, обработка пропусков, преобразование признаков и анализ зависимостей.

Полученные данные были подготовлены для дальнейшего анализа и построения моделей машинного обучения. Выявлены ключевые факторы, влияющие на выживаемость пассажиров, такие как пол, класс билета и размер семьи.

Выводы

  1. Данные требуют предварительной очистки перед анализом
  2. Заполнение пропусков медианой является эффективным методом
  3. Создание новых признаков повышает информативность данных
  4. Наибольшее влияние на выживаемость оказывают пол и класс билета
  5. Pandas является удобным инструментом для анализа данных
