## Описание набора данных  
В работе использован датасет Titanic с платформы Kaggle. Данные содержат информацию о пассажирах корабля, включая:

- PassengerId — идентификатор  
- Survived — факт выживания  
- Pclass — класс билета  
- Name — имя пассажира  
- Sex — пол  
- Age — возраст  
- SibSp — количество родственников (братья/сёстры/супруги)  
- Parch — количество родителей/детей  
- Ticket — номер билета  
- Fare — стоимость билета  
- Cabin — каюта  
- Embarked — порт посадки
